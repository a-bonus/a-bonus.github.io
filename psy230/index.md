# PSY 230: Introduction to Statistics

This site provides resources and guides for ASU's PSY 230 - Introduction to Statistics course. Here you'll find detailed guides and materials to help you understand and apply statistical concepts.

## How to Use This Site

This site is designed to complement your course lectures and discussions. Each module contains:

- Step-by-step guides
- Critical thinking prompts
- Practical hints for statistical software
- Detailed explanations of statistical concepts

The guides are written in a documentation style to help you understand the _why_ behind statistical procedures, not just the _how_.
